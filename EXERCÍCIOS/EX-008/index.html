Principais Formatações

Negrito / Desrtaque

Nesta frase temos um Negrito: Usando a tag <b> (Não Semântica)

Nesta frase temos um Negrito: Usando a tag <strong> (Semântica)

Itálico / Ênfase

Nesta frase temos um Itálico: Usando a tag <i> (Não Semântica)

Nesta frase temos um Itálico: Usando a tag <em> (Semântica)

Texto Marcado

Podemos criar também um texto marcado com <mark>

Podemos criar também um texto marcado mas com cores diferentes usando <mark style="background-color: blue;">

Podemos criar também um texto marcado mas com cores diferentes usando <mark style="background-color: rgb;">

Textos Grandes e Pequenos

Estamos criando um texto GRANDE e um texto PEQUENO, neste parágrafo.

Textos Deletados

Podemos marcar um texto excluído para indicar que ele deve ser lido, mas não considerado: Para isso usamos a tag <del>

Textos Inseridos

Podemos marcar um texto inserido para dar uma ênfase e indicar que ele foi adicionado: Para isso usamos a tag <del>

Existe também o sublinhado com a tag <u> (NÃO SEMANTICA)

Textos Sobrescritos

Para escrever coisas do tipo: x20+3: Para isso usamos a tag <sup>

Textos Subscrito

Para inserir coisas do tipo H2O: Para isso usamos a tag <sub>
